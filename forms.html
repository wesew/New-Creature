New Creature Counseling | Forms

For Clients at the Lebanon, TN clinic, download, print, fill out, and return the following forms:

  1. Appropriate Background Survey- If the identified client is an adult, please print the “Client Background Survey.” If the identified client is a couple, please have each spouse fill out his/her own survey. If the identified client is a child, please print the appropriate parent survey (either Age 1-4 or Age 5-16), as well as a “Client Background Survey” for the parent attending the initial session.
  2. “Feeling Scale”- Please print one for each client attending the session.
  3. “Personal Information Form”- Please print one for each client attending the session.
  4. “Private Practice Disclosure Statement.” It is only necessary to print one disclosure statement, even if there are multiple individuals attending the session.

For additional information, contact Janet Mann: 615.318.0572
